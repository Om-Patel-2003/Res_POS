🚀 Restaurant POS System
🔒 POS: End-to-end restaurant point of sale system with role-based staff access, smart dish selection, and Razorpay payments.

✅ Robust Security: Fine-grained staff authorization for safe, reliable operations.

⚡ Fast Order Workflow: Intuitive, efficient UI for quick order placement and billing.

💳 Real Payments: Integrated Razorpay for secure, real-world payment flows.

🛠️ Scalable Codebase: Clean, maintainable architecture built for growth.

🎯 Business-Driven: Solves real restaurant needs with practical, user-friendly design.
