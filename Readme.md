

🍽️ Restaurant POS (Point Of Sale) System

This project demonstrates a simple Restaurant POS System with:

✅ Role-Based Staff Authorization

🍲 Dish Selection by Restaurant Staff

💳 Razorpay Payment Integration (Test Mode)

📌 1. Role-Based Staff Authorization

Staff members can log in with different roles and permissions to manage the restaurant operations securely.


View Image

🍴 2. Choosing Favourite Dishes

Restaurant staff can select and manage favourite dishes for customers.


View Image

💳 3. Razorpay Payment Integration

Secure payment integration using Razorpay Test Mode for processing online orders.
