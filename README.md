## 🛠 Features

- 🔐 Authentication system using JWT
- 📦 Modular backend structure (Node.js / Express.js)  
- 🧾 Order & billing management  
- 📊 Real-time order updates  
- 💰 Razorpay integration for payments  
- 📡 RESTful API design  
- 🧩 Easy to integrate with frontend
